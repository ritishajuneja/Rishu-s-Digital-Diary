# Rishu's Digital Diary 📖✨

Rishu's Digital Diary is a simple Python-based journal application that allows users to write and read personal diary entries.

The application uses Python File I/O to store entries permanently, making it possible to revisit thoughts, memories, and experiences anytime.

---

## Features

- Write diary entries
- Read saved diary entries
- Mood tracking
- Date-based journaling
- Permanent storage using text files
- Beginner-friendly console application

---

## Concepts Used

- Variables & Data Types
- User Input
- Conditional Statements
- File Handling (File I/O)
- Basic Data Storage
